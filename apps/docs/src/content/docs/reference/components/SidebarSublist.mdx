---
title: "SidebarSublist"
---

Ion's `SidebarSublist` override allows you to use Icons in the sidebar.

import { Code } from "@astrojs/starlight/components";
import sidebarSublistComponent from "starlight-ion-theme/components/SidebarSublist.astro?raw";

<Code
  code={sidebarSublistComponent}
  lang="astro"
  title="SidebarSublist.astro"
/>
